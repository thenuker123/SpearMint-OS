# 🍃 SpearMint OS (v1.0.0)
> A custom hybrid operating system combining **Linux Mint** and **Kali Linux**, fine-tuned for ultimate gaming performance and development workspace agility.

## 🚀 Key Feature: "Flip the Switch"
SpearMint features a dedicated desktop utility launcher that allows users to alter underlying kernel behaviors instantly:

*   **🎮 Gaming Mode:** Unloads active RAM blocks, halts coding resource containers, sets CPU power tracking to maximum performance, and optimizes frame outputs.
*   **💻 Developer Mode:** Resumes secondary network modules (like Docker), returns system resource balances to standard eco thresholds, and alerts development toolsets.

## 📦 Packages Included
*   **Desktop:** Linux Mint XFCE (Sleek and Light)
*   **Security Architecture:** Kali Rolling Network Utilities (`kali-linux-core`)
*   **Gaming Runtime:** Feral GameMode, Wine64 API translation layer, Proton/Steam system preparation
*   **Development Compilers:** Git tracking, Python environment isolation, Node.js packages, Docker systems
